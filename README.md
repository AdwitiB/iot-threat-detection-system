# 🛡️ IoT-Based Threat Detection System for Surveillance Network

> PGDM Major Project | Information Technology | Atharva School of Business (AICTE Approved)

---

## 📖 Overview

This project was developed as part of my Post Graduate Diploma in Management (Information Technology).

The project explores how Internet of Things (IoT), Artificial Intelligence (AI), and Cybersecurity can be integrated to improve surveillance systems through intelligent threat detection and real-time monitoring.

The objective was to analyze current surveillance challenges and propose a scalable, technology-driven solution for smarter and more efficient security systems.

---

## 🎯 Problem Statement

Traditional surveillance systems rely heavily on manual monitoring, making them time-consuming and prone to delayed responses.

This project investigates how IoT devices and AI-driven threat detection can improve surveillance efficiency while enhancing security, automation, and operational performance.
